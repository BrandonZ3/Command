# Command

A custom interpreter that parses and executes predefined commands.
It can also execute scripts.

[CPPLib Repo](https://github.com/BrandonZ3/CPPLib) is a dependency of this project.

---

## Getting Started
1. Clone the repo  
2. Build in Visual Studio  
3. Run the executable  
4. Enter commands in the console  

To see all the commands that are available and how they work type 
"h" or "help" and press "Enter".
